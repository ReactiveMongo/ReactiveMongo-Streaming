# ReactiveMongo Streaming

**Modules:**

- [Akka Stream](./akka-stream/README.md)
- [Iteratees](./iteratees/README.md)

## Build manually

The Akka Streams extension for ReactiveMongo can be built from this source repository.

    sbt publishLocal

To run the tests, use:

    sbt test
